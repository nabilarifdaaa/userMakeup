## Deskripsi
Admin Wardah Brand Makeup
- Aplikasi berbasis android yang menampilkan list produk,user,rating yang ditujukan untuk admin. 
- Di Aplikasi ini bisa menambahkan, mengedit, dan menghapus produk,user,rating
