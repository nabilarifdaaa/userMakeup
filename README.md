# Wardah Brand Makeup
| Nabila Rifda R | 20 |


## Deskripsi
Wardah Brand Makeup
- Aplikasi berbasis android yang untuk melihat katalog makeup yang ditujukan untuk user. 
- Di Aplikasi ini bisa login, register, melihat produk berdasarkan kategori
